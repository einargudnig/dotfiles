# verify-pro lessons

Sidecar file. Verify-pro appends YAML-frontmatter entries here when it gets stuck.
Each entry is a self-contained lesson the skill loads on future runs.

Format reference: see `SKILL.md` § "Recording a Lesson (on stuck)".

Manual edits welcome. To retire a lesson, set `confidence: 0.0` or add `archived: true`.
To delete entries, see `prune.md` for the audit workflow first.
